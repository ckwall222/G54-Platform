## Design Principles

1. **Clarity over cleverness** — Every screen should be immediately understood by its intended persona.
2. **Role-appropriate density** — Finance and production users need data-dense views; executive users need high-level summaries.
3. **Consistent feedback** — Every action should have a visible, immediate response.
4. **Accessible by default** — Color alone never conveys meaning; contrast ratios meet WCAG 2.1 AA minimum.
